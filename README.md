# PUBG DashBoard


# 1. GIT 내려받고 반드시 해야할 명령어
```
npm install (package.json참조해서 자동으로 인스톨한다.)
```

# 2. api키 수정
```
setting.properties
```

# 3. 테스트방법
```
npm test (mocha모듈을 사용해서 test.root.js를 실행한다 [package.json scripts안에 test스크립트 있음])
```

# 4. web 서버실행
```
npm start
```

# 5. web 서버접속
```
http://localhost:2000/
```
